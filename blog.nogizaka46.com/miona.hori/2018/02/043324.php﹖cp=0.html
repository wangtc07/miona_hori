2018/0221Wed堀未央奈おやすみアドレナリン。

 
 
 
 
ブリューゲル展に行ってきました。
モーニング娘。の飯窪春菜ちゃんと！
 
ブリューゲル一族の繋がりや
受け継がれた絵、想いが溢れた
素敵な空間でした。
 
私はヤンブリューゲル2世の画風が
お気に入りでした✨
花の描かれ方がとっても鮮やかなの
昨日描かれた絵？って思うくらい
新鮮で明確☺︎
皆さんも是非...！
 
私にとって美術館と映画館は
息抜き且つエネルギー源となる
場所なの
 
チャージチャージ( ᵒ̴̶̷̤໐ᵒ̴̶̷̤ )
 
 
 
その後は初めて、アメ横に行ったり
ハンバーグも食べました
楽しかったです♪
 
 
 
 
 
先日はarの撮影で
春を感じてきました〜
 
でもまだ肌寒い毎日が続いてますね...
早く暖かくならんかなぁ。
 
ストーブだと思ってあったかい〜って
暖をとっていたら全然火付いてないただの
置物で、わたし自身に太陽が当たっていたから
あったかかったっていう話。
恥ずかしい話。
春菜ちゃんに 太陽当たってるからだよ！
と大いにつっこんでいただいて（ ｉ _ ｉ ）
気づきましたありがとう
 
 
 
かわいいバッグ。
お気に入り。

質問返しブログ
 
待っててください！
 
今日はレコメン
聴いてねー( •̤ᴗ•̤ )

コメントいただきますがたまに載せてる

おうち時間を利用して

話題として家でいろんなメイクをして
写真撮ったりしてます

でも勘違いされるのもいやなので
しばらくやめます




ということで
みなさんのおすすめの曲、MV
を教えてください
いろんなアーティストさんのMVを観て
私なりの解釈や感想をあげていきたいとおもいます
勉強になりそうだ〜(｡･ω･｡)！


ではは

メンバーに会いたいよ( ; _ ; )
みなさんにも( ; _ ; )

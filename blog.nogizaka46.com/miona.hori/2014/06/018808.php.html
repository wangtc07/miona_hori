2014/0605Thu堀未央奈夢の国フロリダへ

 
 
 
 
 
 
こんばんは☆
今日はお昼に高山さんとご飯を
食べに行ってきました！
堀未央奈です！
楽屋の席の隣は日奈子です〜
正解者結構いたなぁ
2人でふざけてると先輩達に
いつも笑われます(笑)
ちなみに今回のブログも長いので
何回かに分けて読むのをオススメ
 
 
 
 
最近は制作期間も落ち着き
プリンシパル公演真っ最中〜
制作期間あっという間でした。
そしてプリンシパル初日で
真夏の全国ツアー2014
が発表されました！！！
2014年8月16日大阪市中央体育館
2014年8月21日福岡国際センター
2014年8月24日仙台ゼビオアリーナ
2014年8月26日日本ガイシホール
2014年8月30日明治神宮野球場
 
 
30日はなんと野外の明治神宮☆
嬉しい〜〜日焼けしちゃうくらい
最高に盛り上がるライブにしたいな☆
サブ地元の名古屋ガイシもあったりと
各地で開催されるので地方の皆さんに
会えるのを楽しみにしています
よろしくお願いします！
日焼けバチコイ！！！！
いや、やっぱり焼けたくないm(._.)m
日焼けどめもちゃんと付けて
万全にして盛り上がりましょう♡♡
 
 
 
聞いて聞いて〜！
金曜ロードSHOW!でジブリ祭りが！
7月4日もののけ姫
7月11日となりのトトロ
7月18日借りぐらしのアリエッティを
やるんだって！！きゃー嬉しいー
夏のジブリ祭りは私にとって本当に
至福の時なんです。
近所の八百屋でスイカ買ってきて
お母さんに切ってもらって
先にお風呂入って冷えたスイカを
食べながらジブリを見るのが、、
ふぉあ。たまらんのです。楽しみ。
 
海がきこえる
コクリコ坂から
魔女の宅急便
耳をすませば
猫の恩返し辺りも見たいな〜
 
ちなみに台本を読んだり集中したい
時のBGMはジブリのサウンドトラック
かディズニーのサウンドトラック！
落ち着くの〜♡。
 
 
 
 
フルーツ大好き人間U^ェ^U
 
 
最近気になる事
みさ先輩の香水の香りがスキすぎて
どこの香水なのか非常時に気になる。
匂いフェチとしてあの香りは100点。
あの香りに自分もかこまれたい。
部屋に撒きたい。そんな事が
気になっている堀でした(笑)
 
 
 
エピソード
カップリングPV撮影の時に
泊まったホテルの相部屋が
井上さん、生駒さん、みなみ、私で
4人部屋だったのでベッドが4つあって
そのうちの1つだけ別室にあって...
(隔離されてる感じの)
私とみなみさんが隣同士のベッドを
一足早くに選んで井上さんは
その近くにあるベッドにしようかな
みたいな感じで、あの部屋は...？？
ってなったから自分から部屋長を
立候補してた生駒さんに私が
 
生駒さん班長ですよね(  ˆᴗˆ  )
あの...班長は...あの部屋で(  ˆᴗˆ  )
決まりです(  ˆᴗˆ  )
 
っていじったらやだぁやだぁって
生駒さんが怖がり始めたので
それを見た私が
 
え、部屋長にさっき
立候補してたじゃないですか(  ˆᴗˆ  )
決まりです(  ˆᴗˆ  )
なんか、あれですよね(  ˆᴗˆ  )
パラノーマルアクティビティに
出てきそうな部屋ですよね(  ˆᴗˆ  )
怖い怖い〜
 
とか言ってたら(爆笑しながら)
私が寝ようとしていたベッドを
生駒さんが奪い取って！！！
結局仲良く2人で寝ました♡笑
素敵な思い出が沢山できた！
いつもはいじられる側だから
時々いじりたくなる！

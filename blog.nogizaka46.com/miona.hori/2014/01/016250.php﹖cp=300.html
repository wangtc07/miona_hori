2014/0117Fri堀未央奈ほっぺを触られる

 
 
 
 
 
 
昨日は撮影終わりにかりんと
買い物に行ってきました〜
洋服の趣味合うしパンケーキ
食べたりして楽しかった♡
 
 
 
 
 
編み編みコートあったかい
 
 
こんばんは
堀未央奈です
 
 
 
 
 
今日のNOGIBINGO2に
2期生も参加させていただいています
内容は1期生vs2期生対決！
それぞれのコスプレをして
登場しますのでお楽しみに
 
 
私の格好は...まだ秘密です
とりあえず恥ずかしかった(笑)
でも私といえばコレだろう
っていう格好なので是非
見ていただきたいな〜♪♪
 
 
 
 
ひなみお
 
 
日テレ26:28〜
是非見てください！
 
 
 
 
 
 
私服
 
 
 
 
 
outer...snidel
tops...snidel
botoms...Lily Brown
shose...oriental traffic
 
 
 
あ、そうそう聞いて下さい！
この間日奈子とお好み焼きを
食べに行ったんですけど時間が
余ったからお洋服見に行こうよって
提案したら寒いじゃんってあっさり
断られました( ; _ ; )
次は絶対に連れて行くもん( ; _ ; )
相変わらず私にだけSです( ; _ ; )
 
 
でも普段は私のわがままを言っても
聞いてくれるし私がスイーツ食べたい
とか服みたいって言うとブツブツ
仕方ないなーと言いながらも
ついてきてくれる所が大好きです♡
 
昨日は夜中まで2人で
ホラー映画みたよ！
怖かったから顔を伏せたら
見なさいって怒られました(笑)
 
 
 
 
 
 
質問返します
 
 
◎未央奈ちゃんがよく見てる
雑誌って何ですか？
 
○雑誌はその時々によるけど
石原さとみさんが出ていたら
必ず買います！
ar、NYLON、装苑、vivi
あと海外雑誌はチェック！
arは雰囲気からもう可愛くて
切り抜きを持ち歩くくらい好き♡
 
 
◎遊園地の絶叫系って乗れる派？
 
○苦手(>_<)夢の国で
スプラッシュマウンテンに乗った時に
怖くて日奈子にしがみついたら
写真に顔が写っていませんでした...
 
 
◎ほんまに暇な時は何する？
 
○休日なら出掛ける
休憩時間なら近くにいる人に
ちょっかいをかける
 
 
◎クレープとパンケーキ
どっちが好き？
 
○チョコバナナクレープも
生クリーム無しフルーツパンケーキも
選べないくらい好き♩
 
 
◎春の高校バレー見てましたか？
 
○はい！
お母さんが昔バレーをしていて
好きなので毎年一緒に見ました
小さい頃からの密かな夢は
お母さんを東京のバレー観戦に
連れていってあげることでした
 
 
◎未央奈は寒い日最高
どれくらい重ね着してる？
 
○最高2枚！自然児！
 
 
◎顔の横にある髪の毛
なんて言うの？
 
○くるくる触覚〜
 
 
 
 
 
 
あと！
 
Amebaブログが回ってきました
私は19日更新！見てねー
コメントもしっかり見ています♡
 
 
 
 
 
昨年の夏の写真
 
純奈に「モロ私のタイプ！」と
絶賛されたので載せておきます
 
 
 
よく岩盤浴に行ったなー
すっぴん！！
 
今年は海いきたひ
 
 
 
 
 
以上。
